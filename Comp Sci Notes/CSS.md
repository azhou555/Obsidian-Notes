### Relative vs AbsoluteğŸ

An absolute measurement unit doesn't change regardless of external settings and will remain the same everywhere. Relative units on the other hand have a baseline of sorts that will impact the size displayed. Taking em for example, if the page's default size is 16px, then a piece of text with size 2em will display 32px. However, if a user instead had a default size of 25px rather than 16px, then the same piece of text would then be displayed as size 50px for them. ### Font Family ğŸ¦‘

Font can make a large difference in the appearance of your websites. However, HTML itself provides a limited number of font including serif, sans-serif, cursive, fantasy, and monospace.
![[Comp Sci Notes/Images/image.png]]
However, you also have access to system fonts including Verdana, Tahoma, Times New Roman, Georgia, and others that are installed universally. One way of implementing this is to simply list out different fonts, and the system will use whichever one works first:
```
font-family: "Gill Sans", Calibri, Verdana, sans-serif;
```
Another option is to use a font provided by Google. To do so, go to <https://fonts.google.com>, find a desired font, copy the <link> tag generated by the site and paste it in the html file somewhere within the <head> tag. Make sure to link the file in prior to the <section> or the link to the CSS <link> tag. Alternatively, you can download the font family and add it to the root directory.

### Specifying a Color

There are several ways of specifying a specific color, but 3 main ones are typically used:

* Color Keyword
	* red, green yellow, etc
	* Notably also contains less known and used colors such as peachpuff, lemonchiffon, etc
* RGB()
	* rgb() is a built-in CSS function that takes 3 values for red, green, and blue, respectively, between 0 and 255 to produce a color.
	* rgb(255, 218, 185) produces PeachPuff
* RGB code
	* 6 digit RGB code in the form rrggbb produces a color
	* rr is a two digit value specifying the red in the color, gg specifying the green, and bb specifying the blue
	* Each 2 digit code goes from 00 to ff represents 255 in hexadecimal
	* PeachPuff is produced by #ffdab9

Extensive list of color can be found at <https://developer.mozilla.org/en-US/docs/Web/CSS/named-color> or <https://htmlcolorcodes.com/color-names/>
We use the color attribute to apply a color to text. The following color styles are identical:
```
color: peachpuff;
color: rgb(255, 218, 185);
color: #ffdab9
```
We can also go as far as to set the background color of an entire page or particular elements through:
```
background-color: peachpuff;
background-color: rgb(255, 218, 185);
background-color: #ffdab9
```
